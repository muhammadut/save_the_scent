# Tech Stack - SaveTheScent

## Frontend Framework
- **Next.js (App Router):** For high-performance, SEO-friendly architecture.
- **TypeScript:** For robust and maintainable code.
- **Tailwind CSS:** For efficient, utility-first styling.

## Animation & Motion
- **GSAP (GreenSock Animation Platform):** Primary engine for complex, timeline-based scroll animations (ScrollTrigger).
- **Framer Motion:** For UI-level interactions, page transitions, and shared element animations (Scent Cards).
- **Lenis:** For smooth, high-end momentum scrolling.

## Media & Assets
- **Self-hosted / Optimized MP4:** High-performance videos stored in the `public/` directory (initial approach).
- **Textures:** CSS grain filters, SVG noise, and WebGL-based grain overlays for artistic elements.

## Communication & Inquiries
- **EmailJS:** For handling the inquiry form submissions and sending automated emails.
- **Direct Phone Integration:** Click-to-call links for immediate inquiries.

## Deployment & Infrastructure
- **Vercel:** Optimized hosting for Next.js with edge functions and global CDN.
